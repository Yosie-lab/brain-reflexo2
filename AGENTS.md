# Nebula Garden（脳リフレクソ2）— Agent 向けガイド

引力で小惑星を星に引き寄せる、目に優しい宇宙アクション。  
脳リフレクソ系譜の Vanilla HTML / CSS / JS（フラット 4 ファイル）。

## 前提

- **第一ターゲットは iPhone Safari**
- フレームワーク・バンドラは入れない（依頼がない限り）
- UI は日本語メイン、フィードバック語は日英ペア。コメントは日本語でよい
- 「やさしく・ゆっくり」が体験の核。難易度や演出を攻略ゲーム寄りにしない

## 構成

```
index.html / style.css / app.js
```

- 設定は `CONFIG` に集約。調整はまずここ
- `'use strict';` とセクション区切り（`// ====...`）の流儀を維持
- 脳リフレクソ由来のパターン（星色、Audio unlock、スワイプバック対策など）を安易に消さない

## 実装ルール

- 変更は最小差分。巨大リファクタやファイル分割は依頼があるまでしない
- SoundEngine の unlock はユーザー操作のコールスタックで呼ぶ前提を守る
- スポーン間隔・同時数・引力などは `CONFIG` 経由で触り、バランス意図のコメントを残す
- タッチとマウス両対応、オーバースクロールや iOS 戻るジェスチャを壊さない
- やり取りは日本語。確認が必要なものを除き、区切りごとに commit して GitHub へ push する

## やってほしくないこと

- 脳リフレクソ本体への無断マージ前提の大改変
- 依存追加、TypeScript 化、ビルド導入
- HUD・スコア・カード UI で画面を賑やかにしすぎること
